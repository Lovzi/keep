```cython
class Solution(object):
    def findContentChildren(self, g, s):
        """
        :type g: List[int]
        :type s: List[int]
        :rtype: int
        """
        g.sort()
        s.sort()
        i = 0
        j = 0
        cnt = 0
        while j < len(g):
            c = g[j]
            if i >= len(s):
                break
            if c <= s[i]:
                cnt += 1
                j += 1
            i += 1
        return cnt
```


# 分析
此代码根据最小匹配原则，从小到大开始匹配，如果不能匹配当前最小食量小朋友的饼干将被丢弃
